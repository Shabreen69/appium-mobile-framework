### 3b. Upload the code using GitHub Desktop (easiest — no command line)
1. Download **GitHub Desktop**: https://desktop.github.com
2. Sign in with your GitHub account
3. Click **"Clone a repository"** → your new repo
4. Open the cloned folder on your computer
5. Copy all files from the provided project folder into it
6. In GitHub Desktop: write commit message → **"Commit to main"** → **"Push origin"**

### 3c. OR upload via browser (no install needed)
1. Open your new repo on GitHub
2. Click **"Add file"** → **"Upload files"**
3. Drag and drop the project folder contents
4. Click **"Commit changes"**
